Copy object person1 to person2

Code

{% highlight js %} var person1 = { name: 'Charles', age: 31, married: true }; var person2 = Ink.extendObj({}, person1); /* person1 will be: { name: 'Charles', age: 31, married: true } person2 will be: { name: 'Charles', age: 31, married: true } */ {% endhighlight %}

Copy object but define default values in new object.

Code

{% highlight js %} var person1 = { name: 'Charles', age: 31, married: true }; var person2 = Ink.extendObj({ hasGlasses: true, married: false }, person1); /* person1 will be: { name: 'Charles', age: 31, married: true } person2 will be: { name: 'Charles', age: 31, married: false, hasGlasses: true } */ {% endhighlight %}
